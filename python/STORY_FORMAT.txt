STORY STRUCTURE FORMAT
======================

Each scene in the STORY dictionary has this structure:

{
    "scene_name": {
        "image": "path/to/image.png or .gif",
        "text": "Story text to display",
        "choices": [
            ("Button label", "next_scene_name"),
            ("Button label", "next_scene_name"),
        ]
    }
}

ADDING FUNCTIONS TO SCENES
===========================

To add a function that runs when entering a scene:

1. Create a new function at the top:
   def func_scene_name():
       # Your code here
       print("Entering scene!")

2. Add it to SCENE_FUNCTIONS dictionary:
   SCENE_FUNCTIONS = {
       "scene_name": func_scene_name,
   }

The function will automatically execute when the scene is displayed.

EXAMPLE
=======

def func_pilseta_1():
    print("Player arrived at city!")
    
SCENE_FUNCTIONS = {
    "start": func_start,
    "pilseta_1": func_pilseta_1,
    "end": func_end,
}

Now when "pilseta_1" scene starts, the function will run first, then display the scene.
